# README

Prepare environment

* Install CLion IDE
* Install dependence libraries

Qt Create with Qt libraries

```
sudo apt-get install \
    qtcreator \
    qtbase5-dev
```

Tested on qt 5.5.1

* Install cmake

```
sudo apt-get install cmake
```

* Install OpenGL

```
sudo apt-get install \
    libgl1-mesa-dev \
    libglu1-mesa \
    libglu1-mesa-dev \
    freeglut3-dev \
    binutils-gold \
    libglu1-mesa:i386
```
